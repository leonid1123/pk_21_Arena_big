## Название проекта

КРОВАВАЯ АРЕНА СТРАШНОЙ СМЕРТИ — текстовая консольная игра на Python

## Краткая концепция

Арена для поединков один на один: игрок (персонаж) против противника. Игра проходит в текстовой консоли, без GUI. Игрок проходит серии боёв (по три боя), получает деньги и может улучшать экипировку и характеристики.

## Игровые сущности

### Персонаж (игрок)

- **Характеристики:**
    - **СИЛА** — влияет на урон.
    - **ЛОВКОСТЬ** — влияет на урон и порядок ходов.
    - **ЖИЗНЬ** — HP; при получении урона уменьшается; при ≤ 0 — персонаж погибает.
- **Экипировка:**
    - **ОРУЖИЕ** — одно активное (КОПЬЁ, МЕЧ, ТОПОР).
    - **БРОНЯ** — одно активное (КОЖАНАЯ, КОЛЬЧУГА, ПОЛНЫЙ ДОСПЕХ).

### Оружие

- **КОПЬЁ** — урон зависит преимущественно от ЛОВКОСТИ.
- **ТОПОР** — урон зависит преимущественно от СИЛЫ.
- **МЕЧ** — урон зависит от СИЛЫ и ЛОВКОСТИ (смешанная шкала).
- Пример формул (рекомендуется зафиксировать при реализации):
    - Копьё: damage = base * (1 + 0.02 * ловкость)
    - Топор: damage = base * (1 + 0.02 * сила)
    - Меч: damage = base * (1 + 0.015 * сила + 0.015 * ловкость)

### Броня

- **КОЖАНАЯ** — поглощает 10% входящего урона.
- **КОЛЬЧУГА** — поглощает 15% входящего урона.
- **ПОЛНЫЙ ДОСПЕХ** — поглощает 50% входящего урона, но требует минимального значения СИЛЫ (например, Strength ≥ X).
- Броня уменьшает итоговый урон после расчёта атакующего урона.

### Враги

- Генерируются случайным образом выбором из 5 типов.
- Обладают теми же характеристиками и вариантами предметов, что и игрок (разные стартовые значения СИЛА/ЛОВКОСТЬ/ЖИЗНЬ и экипировка).
- Могут иметь уникальные способности (опционально): вампиризм, оглушение, щит и т.п.

## Игровой цикл и механика боя

### Общий цикл

1. Игрок проходит серию из трёх боёв подряд.
2. После каждого поединка ЖИЗНЬ игрока полностью восстанавливается (перед следующим боем персонаж выходит на арену с полной ЖИЗНЬЮ).
3. За победу над каждым врагом игрок получает случайное количество денег.
4. После трёх боёв игрок может потратить деньги в магазине на покупку предмета (оружие или броня) или улучшений характеристик.

### Порядок ходов

- Порядок атак в поединке определяется сравнением ЛОВКОСТИ:
    - Если ЛОВКОСТЬ игрока > ЛОВКОСТЬ врага → игрок наносит удар первым; если противник остаётся жив, отвечает.
    - Если ЛОВКОСТЬ игрока < ЛОВКОСТЬ врага → враг наносит удар первым аналогично.
    - Если ЛОВКОСТЬ равна → удары происходят одновременно (оба наносят урон; затем вычитается нанесённый урон из ЖИЗНИ).
- При последовательных ударах проверка смертности выполняется после каждого отдельного удара (тот, кто остаётся жив — отвечает по правилу очередности).

### Расчёт урона (порядок операций)

1. Вычисляется базовый урон атакующего в соответствии с типом оружия и характеристиками (формулы как выше).
2. Применяются дополнительные модификаторы: критический удар (опционально), специальные эффекты оружия/врага.
3. Если удары происходят одновременно, оба расчёта выполняются независимо перед вычитанием HP.
4. Применяется броня защищающего: итоговый полученный урон уменьшается на процент, заданный бронёй.
5. ЖИЗНЬ цели уменьшается на полученный урон. Если ЖИЗНЬ ≤ 0 — цель погибает.

## Экономика

- Награда за победу: случайное количество денег (диапазон и распределение следует задать при реализации; например, 10–50 монет).
- Магазин: предметы (ОРУЖИЕ и БРОНЯ) покупаются за деньги; можно добавить требования по характеристикам (например, полный доспех требует СИЛЫ ≥ 15).
- Опции покупки: новое оружие, доспех, расходники, улучшения характеристик, восстановление между сериями (если изменить правило).

## Техническая структура (файлы / модули)

- main.py — точка входа, игровой цикл, меню.
- player.py — класс Player (характеристики, экипировка, методы атак/получения урона).
- enemy.py — генератор и класс Enemy (унаследовать от Player или общий базовый класс).
- shop.py — логика покупки/продажи.
- utils.py — вспомогательные функции (рандом, вывод в консоль, сохранение).
- game.sqlite - база данных для хранения информации
	- таблица с характеристиками оружия
	- таблица с характеристиками брони
	- таблица с характеристиками врагов
